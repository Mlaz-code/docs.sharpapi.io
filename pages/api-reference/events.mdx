# Events

Get a list of events with filtering, pagination, and optional odds data.

```
GET /api/v1/events
```

## Authentication

Requires API key. Available to all tiers.

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `sports` | string | No | all | Comma-separated sports |
| `leagues` | string | No | all | Comma-separated leagues |
| `live` | boolean | No | false | Only live events |
| `include_odds` | boolean | No | false | Include odds in response |
| `limit` | integer | No | 50 | Results per page (max 200) |
| `offset` | integer | No | 0 | Pagination offset |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp |
| `pagination` | object | Pagination info |
| `filters` | object | Applied filters |
| `events` | array | List of events |

### Event Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Event identifier |
| `externalIds` | object | IDs from each sportsbook |
| `sport` | string | Sport identifier |
| `league` | string | League name |
| `homeTeam` | string | Home team name |
| `awayTeam` | string | Away team name |
| `startTime` | string | ISO 8601 start time |
| `status` | string | Event status |
| `isLive` | boolean | Whether currently live |
| `liveScore` | object | Score if live (optional) |
| `markets` | string[] | Available market types |
| `bookCount` | number | Sportsbooks with odds |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/events?sports=nba,nfl&limit=20" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/events?sports=nba,nfl&limit=20',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/events',
    params={'sports': 'nba,nfl', 'limit': 20},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```
