# Value Bets

Get enhanced +EV opportunities with confidence analysis and edge calculations.

```
GET /api/v1/value-bets
```

## Authentication

Requires API key. **Pro tier or above required.**

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `sports` | string | No | all | Comma-separated sports |
| `books` | string | No | soft | Target sportsbooks |
| `min_ev` | number | No | 2.0 | Minimum EV percentage |
| `min_kelly` | number | No | 0.5 | Minimum Kelly percentage |
| `limit` | integer | No | 50 | Max results (max 200) |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Request status |
| `timestamp` | string | ISO 8601 timestamp |
| `summary` | object | Summary statistics |
| `count` | number | Results returned |
| `valueBets` | array | Value bet opportunities |

### Summary Object

| Field | Type | Description |
|-------|------|-------------|
| `total` | number | Total opportunities found |
| `avgEV` | number | Average EV percentage |
| `avgKelly` | number | Average Kelly percentage |
| `highConfidence` | number | High confidence count |
| `mediumConfidence` | number | Medium confidence count |
| `lowConfidence` | number | Low confidence count |

### Value Bet Object

| Field | Type | Description |
|-------|------|-------------|
| `eventId` | string | Event identifier |
| `eventName` | string | Event name |
| `sport` | string | Sport |
| `league` | string | League |
| `marketType` | string | Market type |
| `sportsbook` | string | Book with value |
| `selection` | string | Selection name |
| `odds` | number | American odds |
| `decimalOdds` | number | Decimal odds |
| `sharpOdds` | number | Pinnacle odds |
| `sharpDecimalOdds` | number | Pinnacle decimal |
| `sharpBook` | string | Sharp reference |
| `fairProbability` | number | True probability (%) |
| `evPercent` | number | Expected value (%) |
| `kellyPercent` | number | Kelly stake (%) |
| `edgePercent` | number | Edge over sharp |
| `combinedScore` | number | Weighted value score |
| `confidence` | string | low/medium/high |
| `marketLiquidity` | string | low/medium/high |
| `timeUntilStart` | number | Seconds to event |
| `detectedAt` | string | Detection timestamp |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/value-bets?sports=nba&min_ev=3.0" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/value-bets?sports=nba&min_ev=3.0',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/value-bets',
    params={'sports': 'nba', 'min_ev': 3.0},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Confidence Ratings

| Rating | Criteria |
|--------|----------|
| **High** | EV >= 5% + Kelly >= 2% + 4+ books |
| **Medium** | EV >= 3% + Kelly >= 1% + 2+ books |
| **Low** | Below medium thresholds |

## Value Bet vs Basic EV

Value Bets endpoint provides additional analysis:

| Feature | /ev | /value-bets |
|---------|-----|-------------|
| EV calculation | Yes | Yes |
| Kelly sizing | Yes | Yes |
| Edge over sharp | No | Yes |
| Combined score | No | Yes |
| Confidence rating | No | Yes |
| Market liquidity | No | Yes |
| Time to event | No | Yes |

Use `/value-bets` for more informed betting decisions.
