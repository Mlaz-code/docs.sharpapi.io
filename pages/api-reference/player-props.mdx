# Player Props

Get player prop odds across sportsbooks.

```
GET /api/v1/markets/player-props
```

## Authentication

Requires API key. **Pro tier or above required.**

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `sports` | string | No | all | Comma-separated sports |
| `event_id` | string | No | - | Filter by event |
| `player` | string | No | - | Filter by player name |
| `limit` | integer | No | 100 | Max results (max 500) |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Request status |
| `timestamp` | string | ISO 8601 timestamp |
| `count` | number | Results returned |
| `filters` | object | Applied filters |
| `playerProps` | array | Player prop odds |

### Player Prop Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Prop identifier |
| `sportsbook` | string | Sportsbook |
| `eventId` | string | Event identifier |
| `sport` | string | Sport |
| `league` | string | League |
| `player` | string | Player name |
| `team` | string | Player's team |
| `propType` | string | Stat category |
| `selection` | string | Over or Under |
| `line` | number | Prop line |
| `odds` | object | Odds values |
| `eventStartTime` | string | Event start |
| `timestamp` | string | Last update |

### Common Prop Types

| Sport | Prop Types |
|-------|------------|
| Basketball | points, rebounds, assists, threes, steals, blocks, pts+reb, pts+ast |
| Football | passing_yards, rushing_yards, receiving_yards, touchdowns, receptions |
| Hockey | goals, assists, points, shots, saves |
| Baseball | hits, runs, rbis, strikeouts, total_bases |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/markets/player-props?sports=nba&player=lebron" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/markets/player-props?sports=nba&player=lebron',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/markets/player-props',
    params={'sports': 'nba', 'player': 'lebron'},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Tips for Player Props

1. **Compare across books**: Lines can vary 0.5-1.5 points
2. **Check matchups**: Defensive rankings affect props
3. **Monitor injuries**: Teammate injuries increase opportunities
4. **Track trends**: Recent performance vs season average
5. **Watch for outliers**: One book off the market consensus
