# Odds

Get a snapshot of current odds from all sportsbooks.

```
GET /api/v1/odds
```

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sports` | string | Comma-separated sports filter. Example: `nba,nfl,nhl` |
| `books` | string | Comma-separated sportsbooks filter. Example: `draftkings,pinnacle` |
| `api_key` | string | API key (alternative to `X-API-Key` header) |


## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp of the response |
| `count` | number | Total number of odds returned |
| `books` | string[] | List of sportsbooks with data |
| `data` | object | Odds grouped by sportsbook |

## Rate Limit Headers

All responses include rate limit information:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Maximum requests per minute |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Unix timestamp when limit resets |


### Example Request


```bash cURL
curl -X GET "https://sharpapi.io/api/v1/odds?sports=nba" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/odds?sports=nba',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/odds',
    params={'sports': 'nba'},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Odds Object Schema

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique odds identifier |
| `sportsbook` | string | Sportsbook ID (e.g., `draftkings`) |
| `eventId` | string | Event identifier |
| `sport` | string | Sport (e.g., `nba`, `nfl`) |
| `league` | string | League name |
| `homeTeam` | string | Home team name |
| `awayTeam` | string | Away team name |
| `marketType` | string | `moneyline`, `spread`, `total` |
| `selection` | string | The selection (team name or Over/Under) |
| `odds.american` | number | American odds (-110, +150) |
| `odds.decimal` | number | Decimal odds (1.909) |
| `odds.probability` | number | Implied probability (0.52) |
| `line` | number? | Spread/total line value |
| `isLive` | boolean | Whether event is live |
