# Odds

Get a snapshot of current odds from all sportsbooks.

```
GET /api/v1/odds
```

## Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sportsbook` | string | all | Filter by sportsbook (e.g., `draftkings`, `pinnacle`) |
| `league` | string | all | Filter by league (e.g., `nba`, `nfl`, `nhl`) |
| `market` | string | all | Filter by market type (e.g., `moneyline`, `spread`, `total`) |
| `live` | boolean | false | Return only live/in-play events |
| `main` | boolean | false | Return only main markets (moneyline, spread, total) |

## Response Format

All responses follow the standard SharpAPI envelope format:

```json
{
  "data": [...],
  "meta": {
    "count": 50,
    "total": 234,
    "pagination": {
      "limit": 50,
      "offset": 0,
      "has_more": true
    },
    "updated": "2026-01-26T12:00:00Z"
  }
}
```

## Rate Limit Headers

All responses include rate limit information:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Maximum requests per minute |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Unix timestamp when limit resets |

## Odds Object Schema

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique odds identifier |
| `sportsbook` | string | Sportsbook ID (e.g., `draftkings`) |
| `sportsbookDisplayName` | string | Human-readable book name |
| `eventId` | string | Event identifier |
| `sport` | string | Sport (e.g., `nba`, `nfl`) |
| `league` | string | League name |
| `homeTeam` | string | Home team name |
| `awayTeam` | string | Away team name |
| `marketType` | string | `moneyline`, `spread`, `total`, `player_prop`, etc. |
| `selection` | string | The selection (team name or Over/Under) |
| `selectionType` | string | `home`, `away`, `over`, `under` |
| `odds.american` | number | American odds (-110, +150) |
| `odds.decimal` | number | Decimal odds (1.909) |
| `odds.probability` | number | Implied probability (0.52) |
| `line` | number? | Spread/total line value |
| `eventStartTime` | string | ISO 8601 event start time |
| `timestamp` | string | When odds were last updated |
| `isLive` | boolean | Whether event is live |
| `status` | string | `upcoming`, `live`, `ended` |
