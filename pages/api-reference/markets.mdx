# Markets

Get available market types and their configurations.

```
GET /api/v1/markets
```

<Callout type="info">
This endpoint is accessible without authentication at 10 requests/minute. Authenticated requests receive higher rate limits.
</Callout>

## Example Request

<Tabs items={['cURL', 'JavaScript', 'Python']}>
  <Tabs.Tab>
```bash
curl -X GET "https://api.sharpapi.io/api/v1/markets" \
  -H "X-API-Key: YOUR_API_KEY"
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript
const response = await fetch(
  'https://api.sharpapi.io/api/v1/markets',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const { data } = await response.json();
console.log('Available markets:', data.map(m => m.id));
```
  </Tabs.Tab>
  <Tabs.Tab>
```python
import requests

response = requests.get(
    'https://api.sharpapi.io/api/v1/markets',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
markets = response.json()['data']
print(f"Available markets: {[m['id'] for m in markets]}")
```

## Market Schema

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique market identifier (use in API filters) |
| `name` | string | Human-readable name |
| `description` | string | Brief description of the market |
| `category` | string | `main`, `prop`, `period`, `live` |
| `hasLine` | boolean | Whether this market has a line value |
| `selectionTypes` | string[] | Possible selection types |
| `sports` | string[] | Sports that support this market |

## Market Categories

### Main Markets

Core betting markets available for most events:

| Market | Description | Line |
|--------|-------------|------|
| `moneyline` | Pick the winner | No |
| `spread` | Point spread / handicap | Yes |
| `total` | Over/Under combined score | Yes |

### Prop Markets

Player, team, and game proposition bets:

| Market | Description | Line |
|--------|-------------|------|
| `player_prop` | Individual player stats | Yes |
| `team_prop` | Team-specific outcomes | Yes |
| `game_prop` | Game-specific outcomes | No |

### Period Markets

First half, first quarter, and other period-based bets:

| Market | Description | Line |
|--------|-------------|------|
| `first_half` | 1H moneyline/spread/total | Yes |
| `first_quarter` | 1Q moneyline/spread/total | Yes |
| `second_half` | 2H moneyline/spread/total | Yes |

## Selection Types

| Type | Used In | Description |
|------|---------|-------------|
| `home` | Moneyline, Spread | Home team wins/covers |
| `away` | Moneyline, Spread | Away team wins/covers |
| `over` | Total, Props | Over the line |
| `under` | Total, Props | Under the line |
| `yes` | Game Props | Event occurs |
| `no` | Game Props | Event does not occur |

## Using Markets in Filters

Filter odds by market type using the `market` query parameter:

```bash
# Get only moneyline odds
curl "https://api.sharpapi.io/api/v1/odds?market=moneyline"

# Get only spread and total (main markets)
curl "https://api.sharpapi.io/api/v1/odds?main=true"

# Get player props only
curl "https://api.sharpapi.io/api/v1/odds?market=player_prop"
```
