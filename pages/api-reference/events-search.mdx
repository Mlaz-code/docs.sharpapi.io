# Event Search

Search for events by team name.

```
GET /api/v1/events/search
```

## Authentication

Requires API key. Available to all tiers.

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `q` | string | Yes | - | Search query (min 2 characters) |
| `live` | boolean | No | false | Only search live events |
| `limit` | integer | No | 20 | Max results (max 100) |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp |
| `query` | string | Search query used |
| `count` | number | Number of matches |
| `events` | array | Matching events |

### Search Result Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Event identifier |
| `sport` | string | Sport |
| `league` | string | League |
| `homeTeam` | string | Home team |
| `awayTeam` | string | Away team |
| `startTime` | string | Start time |
| `isLive` | boolean | Live status |
| `matchedTeam` | string | Team that matched query |
| `bookCount` | number | Books with odds |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/events/search?q=lakers" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/events/search?q=lakers',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/events/search',
    params={'q': 'lakers'},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Search Tips

- Search is case-insensitive
- Partial matches work (e.g., "laker" matches "Lakers")
- Search matches against both home and away team names
- Use specific team names for better results
