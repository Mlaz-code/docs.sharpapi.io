# Event Details

Get detailed information for a single event including all odds.

```
GET /api/v1/events/{eventId}
```

## Authentication

Requires API key. Available to all tiers.

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `eventId` | string | Yes | Event identifier |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp |
| `event` | object | Event details |
| `markets` | object | Odds grouped by market type |
| `oddsCount` | number | Total odds returned |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/events/evt_nba_lal_bos_20240115" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/events/evt_nba_lal_bos_20240115',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/events/evt_nba_lal_bos_20240115',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```
