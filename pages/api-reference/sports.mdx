# Sports

Get a list of all available sports with event counts.

```
GET /api/v1/sports
```

## Authentication

Requires API key. Available to all tiers.

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp |
| `sports` | array | List of sports with metadata |

### Sport Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Sport identifier (lowercase) |
| `name` | string | Display name |
| `leagues` | string[] | Leagues within this sport |
| `eventCount` | number | Total events with odds |
| `liveEventCount` | number | Currently live events |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/sports" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/sports',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/sports',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```
