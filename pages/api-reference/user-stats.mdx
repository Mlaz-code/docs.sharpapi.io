# User Stats

Get API usage statistics for your account.

```
GET /api/v1/user/stats
```

## Authentication

Requires API key. Available to all tiers.

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Request status |
| `timestamp` | string | ISO 8601 timestamp |
| `stats` | object | Usage statistics |
| `rateLimit` | object | Current rate limit status |
| `features` | string[] | Available features |
| `upgrade` | object | Upgrade info (if applicable) |

### Stats Object

| Field | Type | Description |
|-------|------|-------------|
| `keyId` | string | Your API key identifier |
| `tier` | string | Subscription tier |
| `usage.requestsToday` | number | Requests today |
| `usage.requestsThisMonth` | number | Requests this month |
| `usage.streamsActive` | number | Active SSE streams |
| `usage.streamsAllowed` | number | Max allowed streams |
| `limits.requestsPerMinute` | number | Rate limit |
| `limits.dataDelay` | number | Data delay (seconds) |

### Rate Limit Object

| Field | Type | Description |
|-------|------|-------------|
| `limit` | number | Requests per minute |
| `remaining` | number | Remaining in window |
| `reset` | string | Reset timestamp |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/user/stats" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/user/stats',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/user/stats',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Monitoring Usage

Use this endpoint to:

1. **Track consumption**: Monitor daily/monthly usage
2. **Avoid rate limits**: Check remaining requests
3. **Manage streams**: See active vs allowed streams
4. **Plan upgrades**: Know when to upgrade tier
