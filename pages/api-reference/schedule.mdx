# Schedule

Get the schedule of upcoming and live events.

## Endpoints

```
GET /api/v1/schedule
GET /api/v1/schedule/live
```

## Schedule - Upcoming Events

Get all upcoming events with optional filtering.

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sport` | string | all | Filter by sport (e.g., `nba`, `nfl`) |
| `league` | string | all | Filter by league |
| `date` | string | today | Filter by date (ISO format or `today`, `tomorrow`) |
| `days` | number | 1 | Number of days to include (max 7) |

### Example Request

<Tabs items={['cURL', 'JavaScript', 'Python']}>
  <Tabs.Tab>
```bash
curl -X GET "https://api.sharpapi.io/api/v1/schedule?sport=nba&days=3" \
  -H "X-API-Key: YOUR_API_KEY"
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript
const response = await fetch(
  'https://api.sharpapi.io/api/v1/schedule?sport=nba&days=3',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const { data } = await response.json();
console.log(`${data.length} upcoming NBA games`);
```
  </Tabs.Tab>
  <Tabs.Tab>
```python
import requests

response = requests.get(
    'https://api.sharpapi.io/api/v1/schedule',

)
data = response.json()['data']
print(f"{len(data)} upcoming NBA games")
```

## Live Events

Get all currently live events.

```
GET /api/v1/schedule/live
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sport` | string | all | Filter by sport |

### Example Request

```bash
curl -X GET "https://api.sharpapi.io/api/v1/schedule/live" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Event Schema

| Field | Type | Description |
|-------|------|-------------|
| `eventId` | string | Unique event identifier |
| `sport` | string | Sport key (e.g., `nba`) |
| `league` | string | League name |
| `homeTeam` | string | Home team name |
| `awayTeam` | string | Away team name |
| `startTime` | string | ISO 8601 start time |
| `status` | string | `upcoming`, `live`, `ended`, `postponed` |
| `venue` | string | Event venue (if available) |
| `broadcast` | string[] | TV broadcast networks (if available) |
| `score` | object | Current score (live events only) |
| `period` | string | Current period (live events only) |
| `clock` | string | Game clock (live events only) |

## Status Values

| Status | Description |
|--------|-------------|
| `upcoming` | Event has not started |
| `live` | Event is in progress |
| `ended` | Event has completed |
| `postponed` | Event has been postponed |
| `cancelled` | Event has been cancelled |

<Callout type="info">
Schedule data is refreshed every 5 minutes. Use the `/events` endpoint for more detailed event information.
</Callout>
