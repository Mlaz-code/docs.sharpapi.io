# Live Events

Get currently live (in-play) events with live scores.

```
GET /api/v1/events/live
```

## Authentication

Requires API key. Available to all tiers.

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `sports` | string | No | Comma-separated sports filter |
| `leagues` | string | No | Comma-separated leagues filter |

## Response

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `timestamp` | string | ISO 8601 timestamp |
| `count` | number | Number of live events |
| `events` | array | Live events with scores |

### Live Event Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Event identifier |
| `sport` | string | Sport identifier |
| `league` | string | League name |
| `homeTeam` | string | Home team |
| `awayTeam` | string | Away team |
| `startTime` | string | Original start time |
| `status` | string | Always `in_progress` |
| `isLive` | boolean | Always `true` |
| `liveScore` | object | Current score |
| `markets` | string[] | Available markets |
| `bookCount` | number | Books with live odds |

### Live Score Object

| Field | Type | Description |
|-------|------|-------------|
| `home` | number | Home team score |
| `away` | number | Away team score |
| `period` | string | Current period (Q1, H2, etc.) |
| `clock` | string | Game clock (5:42, etc.) |

### Example Request

```bash cURL
curl -X GET "https://sharpapi.io/api/v1/events/live?sports=nba" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://sharpapi.io/api/v1/events/live?sports=nba',
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://sharpapi.io/api/v1/events/live',
    params={'sports': 'nba'},
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
data = response.json()
```

## Period Formats by Sport

| Sport | Periods |
|-------|---------|
| Basketball | Q1, Q2, Q3, Q4, OT, OT2 |
| Football | Q1, Q2, Q3, Q4, OT |
| Hockey | P1, P2, P3, OT, SO |
| Baseball | T1, B1, T2, B2, ... T9, B9 |
| Soccer | H1, H2, ET1, ET2 |
| Tennis | Set 1, Set 2, Set 3 |
